# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-08

### Added
- Initial public release
- Batch Markdown to Word conversion via Pandoc
- Multi-format export: .docx, PDF, HTML, RTF
- Dark theme Apple-style PySide6 GUI
- Drag-and-drop file/folder support
- Built-in Markdown preview via QtWebEngine
- Custom Word template support
- Progress bar, status indicators, and logging system
- Configuration persistence
